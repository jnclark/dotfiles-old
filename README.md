## muttrc
you need to add in user info, such as email, etc.,
and any aliases you might wish to use

### user
```
# User config
set my_realname="First Last"
set my_username="example"
set my_domain="example.edu"
```

### aliases
```
alias name First Last <example@example.edu>
```
